## ğŸ Usage

Clone the repository and go to the root directory

```bash
$ git clone https://github.com/viniciusgugelmin/ntt-movie

$ cd ntt-movie
```

Install the dependencies

```bash
$ npm install
```

Create a `.env` file in the src directory and add the variables in the `.env.example` file

Run the application:

```bash
$ npm run start
```
